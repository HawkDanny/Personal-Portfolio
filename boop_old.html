Danny Hawk's Boop Game

Danny Hawk's Boop Game is an examination into 'competitive sharing', which is what happens when players are forced to compete against each other while sharing a control. This project started as a web game that can be played here, and has since expanded to become a unity game with custom built arcade controls.
